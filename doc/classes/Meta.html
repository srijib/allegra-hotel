        Drop the current meta in the database immediately

        drop(bool $force) : void

        Parameters

        $force

        bool

        (default FALSE) If set to true, this function will drop all records related to this meta in the OBJECT_has_meta table, and then drop the meta itslef in the meta table. If set to false, this function will only remove the meta in the meta table if no OBJECT is linked to this meta.
